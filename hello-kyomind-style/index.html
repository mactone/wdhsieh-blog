我的新部落格上線了

2026-03-28 筆記

文章目錄

  1. 1. 目前架構
  2. 2. 為什麼這種架構值得用
  3. 3. 下一步

這個站是以 Hexo 建立，並參考 blog.kyomind.tw 的整體風格與部署思路搭起來的。

目前架構

  • 靜態站產生器：Hexo
  • 主題：Even
  • 可部署位置：GitHub Pages / Hetzner / Netlify / Cloudflare Pages
  • 可加裝分析：Umami / GA4
  • 可加裝留言：utterances / giscus

為什麼這種架構值得用

  1. 成本低
  2. 維護簡單
  3. 載入快
  4. 文章以 Markdown 管理，適合長期寫作

下一步

接下來只要補：

  • 網站名稱
  • 作者資訊
  • 網域
  • 部署目的地
  • 分析與留言系統

就可以正式上線。
